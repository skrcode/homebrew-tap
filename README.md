# Skrcode Tap

## How do I install these formulae?

`brew install skrcode/tap/<formula>`

Or `brew tap skrcode/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "skrcode/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
